| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |------------- |--------- |
| direction     | 滚动方向   | String  |   vertical,horizontal       |    vertical    |
| autoplay     | 播放时长   | Number  |   -       |    2000    |
