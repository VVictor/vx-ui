# Spinner旋转

```
<template>
  <div>
    <spinner></spinner>
    </br>
    </br>
    </br>
    <spinner :size="1">
      <span style="line-height:1rem;">LOGO</span>
    </spinner>
  </div>
</template>
</script>
```

#### Props
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |------------- |--------- |
| size     | 大小，单位rem   | Number  |   -       |    0.4    |
